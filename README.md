# pyBybitMarketMaker
Simplified Bybit market maker for personal customisation. Responds to order fills. You need to set initial orders manually.

# Pre-requisites
Bybit account - https://www.bybit.com/app/register?ref=JQoVo

Python - https://www.python.org/

# Install dependencies
pip install bybit-ws

pip install bybit

# Starting the script
Edit your own api key and secret settings. Then start script with:

python pyBybitMarketMaker.py
